# 2DMinecraft

2D Minecraft - is a very simple minecraft, but in 2D. The game contains a day/night cycle, mining and building. It's console app.

# Controls

W, A, S, D - Movement
Q - Change mode (movement, mining and building)
E - Change item (wood, stone)
ESC - Save and exit
